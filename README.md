# 📱 SmartNews App

SmartNews is an AI-powered mobile application designed for students to stay updated with current affairs in a smart and engaging way.

## 🚀 Features

* 📰 Categorized News Feed (Sports, Tech, Education, Politics)
* 🧠 AI-Generated Quiz from News Articles
* ⏱️ Daily 5-Minute News Capsule
* 📊 Performance Tracking
* 🔖 Bookmark Important News

## 🛠️ Tech Stack

* Flutter (Frontend)
* Firebase (Authentication & Database)
* AI APIs (for quiz generation & summaries)

## 🎯 Goal

To make news reading more interactive and useful for students preparing for exams.

🚧 Currently in development (Frontend phase)
